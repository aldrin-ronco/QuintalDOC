===============
Otros productos
===============

		Se usa una misma interfaz 6.6 para dar entrada y salida a un producto de esta categoría. Venta de materia prima o dar salida a un residuo.

Pasos para realizar una entrada de otros productos
==================================================


		1. `Realice una entrada de vehículo lleno <../despachos/frm_despacho_otros_productos.html#registrar-ingreso-de-vehiculo-con-producto>`_
		2. `Realice una salida de vehículo vacío <../despachos/frm_despacho_otros_productos.html#registrar-una-salida-de-vehiculo-vacio>`_

Pasos para realizar una salida de otros productos
=================================================

		1. `Realice una entrada de vehículo vacío <../despachos/frm_despacho_otros_productos.html#registrar-una-entrada-de-vehiculo-vacio>`_
		2. `Realice una salida de vehículo lleno <../despachos/frm_despacho_otros_productos.html#registrar-salida-de-vehiculo-con-producto>`_
